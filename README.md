# Welcome to PowerFit

## Project Info

**Project Name:** PowerFit  
**Live URL:** https://powerfit-puce.vercel.app/

PowerFit is a modern fitness and workout platform designed to help users achieve their fitness goals such as weight loss, weight gain, and muscle building. The platform provides structured workout plans, diet guidance, and a clean, user-friendly experience.

---

## How can I edit this code?

You can edit and manage this project in multiple ways:

### Use Your Preferred IDE (Local Development)

If you want to work locally using your own IDE, follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GITHUB_REPO_URL>

# Step 2: Navigate to the project directory
cd powerfit

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
````


Make sure **Bun** is installed on your system.  
Install Bun from here: https://bun.sh/docs/installation

---

### Edit Files Directly on GitHub

- Open your repository on GitHub
- Navigate to the file you want to edit
- Click the **Edit (✏️)** button
- Make changes and commit them

Changes pushed to GitHub will automatically deploy on Vercel (if connected).

---

### Use GitHub Codespaces

- Go to your GitHub repository
- Click the **Code** button
- Open the **Codespaces** tab
- Click **New Codespace**
- Edit files, then commit and push your changes
